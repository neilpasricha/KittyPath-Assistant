Overview

Simplify and automate the most complicated features of the dated LionPath system. Our extension is meant supplement the course scheduling system provided by LionPath.

Details

school
View a summary of your degree audit
KittyPath technology will read your LionPath degree audit once, and persist it, all while scanning for the requirements you need to graduate. View the most important parts of your audit without spending minutes scanning throught the long file itself. Built in GPA calculator functionality to predict and set your GPA goals.
find_in_page
Instantly search for available classes to schedule
Search for courses based on the name, course code, or requirements that you need to satisfy. Save these courses to view them later and plan your schedule.
dvr
View recommended academic plan
Display the courses you should take based on major and semester. KittyPath will also tell you which courses are available for the chosen semester, as well as redirect you to save that course for later.

Technology

KittyPath is built using JavaScript/Jquery and CSS for the latest version of Chrome, utilizing Chrome Storage API's. Data was acquired using jQuery for user acquired data, and Python's BeautifulSoup4 for large data objects. All of our code is open source.
